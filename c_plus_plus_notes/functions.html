Functions

While our code is being executed each line is usually read one at at time from top to bottom of our file each step one at a time.

If we want to reuse a piece of code we could copy and paste it into our new section; however, this is not ideal for a couple of reasons which I will get into later

There are a couple of ways we can reuse code - one way we have talked about already is in the form of loops. Imagine if we did not have this capability! We would have to copy and paste the same code however many times we wanted an action to happen. Infinite input would be impossible and the frustration is over 9000 if we had a typo in a section that was copied many times!

But what if we want have a piece of code that we use in many places but not in a consecutive fashion. Say I have some code that converts F to C or maybe a US state name to its associated abbreviation. It would be silly to have to copy and paste this method every time we want to use it: ENTER FUNCTIONS!!!
